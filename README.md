# NFTamina

Sitio web estático de presentación para **NFTamina**, una comunidad enfocada en criptomonedas y NFTs.

## Descripción

NFTamina es una landing simple e informativa pensada para presentar una comunidad de entusiastas del ecosistema cripto. El sitio incluye una introducción general, una sección “Sobre Nosotros”, listado de servicios, presentación del equipo y un bloque de contacto.
